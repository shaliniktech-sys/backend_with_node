# Introduction to Node.js

-> JavaScript is a programming language primarily used to add interactivity to web pages.

-> Every browser comes with a JavaScript Engine that executes JavaScript code.

-> Earlier, JavaScript could only run inside web browsers.

-> Some popular JavaScript engines:
   1. Chrome  : V8
   2. Firefox : SpiderMonkey
   3. Safari  : JavaScriptCore

-> Ryan Dahl created Node.js in 2009 by combining Google's V8 JavaScript Engine with C++ libraries.

-> Node.js allows JavaScript to run outside the browser, giving it access to operating system features like:
   - File System
   - Network
   - Processes
   - Memory
   - etc.

-> With Node.js, we can build:
   - Web servers
   - APIs
   - CLI applications
   - Real-time applications
   - Automation scripts

-> Node.js is NOT a framework.
-> Node.js is NOT a library.
-> Node.js is a JavaScript Runtime Environment.
