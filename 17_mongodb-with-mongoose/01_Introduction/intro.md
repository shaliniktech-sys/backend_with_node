# What is MongoDB?

MongoDB is a NoSQL database.

Instead of storing data in rows and columns like MySQL, MongoDB stores data as documents.

Example

```json
{
    "name":"Shubh",
    "age":19
}
```

Documents are stored inside Collections.

Collections are stored inside Databases.

Hierarchy

Database
↓
Collection
↓
Document

MongoDB is

- Fast
- Flexible
- Scalable
- Document Oriented

MongoDB stores data in BSON (Binary JSON).
