# 06_HTTP_Server_NodeJS

A beginner-friendly Node.js project that demonstrates how to create a basic HTTP server using the built-in `http` module.

## What this project covers

- Creating an HTTP server
- Handling incoming client requests
- Sending responses to the browser
- Routing using `req.url`
- Logging every request into a file
- Using asynchronous file handling with `fs.appendFile()`
- Running the server on a custom port

## Features

- Home route (`/`)
- About route (`/about`)
- Default 404 response
- Automatic request logging in `log.txt`

## Tech Used

- Node.js
- HTTP Module
- File System (fs) Module
