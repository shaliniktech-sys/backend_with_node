# 01 - Introduction to Node.js

## About

This project contains my notes and basic concepts from learning **Node.js**. It covers what Node.js is, why it was created, how it works, and how it enables JavaScript to run outside the browser.

## Key Concepts

- JavaScript is a programming language mainly used to make web pages interactive.
- Every browser has its own JavaScript engine to execute JavaScript code.
- Before Node.js, JavaScript could only run inside web browsers.
- Node.js was created by **Ryan Dahl** in **2009** by combining Google's **V8 JavaScript Engine** with C++ libraries.
- Node.js allows JavaScript to access system resources such as the file system, network, processes, and memory.
- It can be used to build:
  - Web Servers
  - REST APIs
  - CLI Applications
  - Real-time Applications
  - Automation Scripts

## JavaScript Engines

| Browser | Engine |
|---------|--------|
| Chrome | V8 |
| Firefox | SpiderMonkey |
| Safari | JavaScriptCore |

## Important

- Node.js is **not a framework**.
- Node.js is **not a library**.
- **Node.js is a JavaScript Runtime Environment.**
