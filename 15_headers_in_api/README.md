# HTTP Headers, API & Status Codes

This project explains the basics of HTTP Headers, APIs, and HTTP Status Codes using Node.js. It includes examples demonstrating how clients and servers exchange information through headers, how APIs work, and how different HTTP status codes are returned based on the request.

## Topics Covered

- HTTP Headers
- APIs
- HTTP Status Codes
- Reading Request Headers
- Sending Response Headers
- Returning Different Status Codes

## Technologies Used

- Node.js
- HTTP Module

## How to Run

```bash
node code.js
```

Open your browser and visit:

```
http://localhost:8000
```

or use Postman.
