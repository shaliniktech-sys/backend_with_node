# HTTP Status Codes

This project demonstrates the use of common HTTP Status Codes in Node.js. Different routes return different status codes to indicate whether a request was successful or if an error occurred.

## Status Codes Used

- 200 OK
- 201 Created
- 400 Bad Request
- 401 Unauthorized
- 403 Forbidden
- 404 Not Found

## Run

```bash
node code.js
```
